JVCalculator
A simple GUI-based calculator application built in Java using the Swing framework. This project demonstrates fundamental concepts of Java GUI development and basic arithmetic operations.

Features
  User-friendly graphical interface.
  Supports basic arithmetic operations:
  Addition (+)
  Subtraction (-)
  Multiplication (*)
  Division (/)
  Clear button (C) to reset the calculator.
  Real-time input and result display.
Project Structure
  Source Code: Java files implementing the calculator's logic and GUI.
  Dependencies: Built with Maven for dependency management.
Technologies Used
  Java: Core programming language for the application.
  Swing: For designing the graphical user interface.
  Maven: For project management and build.
